### 문제 설명

<p>메이플스토리에는 방어율 무시라는 수치가 있다. 특정 보스를 잡기 위해서는 특정 방어율 무시 수치가 되어야 데미지가 정확히 들어가는 시스템으로 이루어져 있다. 물약 아이템을 사용하여 방어율 무시 수치를 증가시킬 수 있다.</p>
