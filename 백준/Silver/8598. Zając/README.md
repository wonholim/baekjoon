 <p>W pierwszym i jedynym wierszu standardowego wyjścia powinna znaleźć się jedna dodatnia liczba całkowita równa minimalnej liczbie skoków, jakie Bajtek musi wykonać, aby dotrzeć do swojej nory, lub słowo "<code>NIE</code>", jeśli dotarcie Bajtka do nory przy użyciu poprawnych skoków nie jest możliwe.</p>
